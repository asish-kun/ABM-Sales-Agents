Introduction

Welcome to the Joone Editor!
This is a FREE application that permits to visually edit a neural network using the core engine's components of Joone.
Visit http://www.joone.org for more informations.

This help system is broken down into the following sections:

  • Tutorial
    This section gives ideas and examples of how to use Joone Edit
  • Reference Information
    This section explains in detail what each graphical component in Joone Editor does
  • Concepts
    Contains a brief explanation of the basic concepts underlying Joone
  • Resources
    Contains some useful links and a glossary of the principal used terms
